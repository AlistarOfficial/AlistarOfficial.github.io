本文概述了 Dolby Atmos Renderer 的不同配置，DAW 和 Renderer 的协同使用，系统时钟和同步，监听杜比全景声内容以及视频重放，敬请查收。

Dolby Atmos Renderer 的不同配置

Dolby Atmos Renderer 软件可通过以下方式运行：

  • 在 DAW 运行的设备上运行
  • 在专门的 Rendering 和 Mastering 工作站中外部运行
  • 在特定的 DAW 中原生使用 Renderer

当 Dolby Atmos Renderer 在专门的外部 Rendering & Mastering Workstation（RMW，渲染和母带工作站）中运行时，可以使用四种平台和 I / O 配置。杜比经销商提供以下配置：

  • 配备 RME MADI 和同步 I / O 的 Windows 10（电脑为自定配置的 Dell 7920）
  • 配备 Focusrite PCIeR Dante I / O 的 Windows 10（电脑为自定配置的 Dell 7920）
  • 配备 Focusrite PCIeR Dante I / O 的 Mac（Mac 硬件最低为 Mini 或者 Pro 规格）
  • 配备 RME MADIface XT I / O 的 Mac（Mac 硬件最低为 Mini 或者 Pro 规格）

在下一个课程中，我们会对这些配置项进行更深入的讨论。

DAW 和 Dolby Atmos Renderer

DAW 是沉浸式音频的创意平台，也是 Dolby Atmos Renderer 中音床、声音对象、OAMD 和 纵向时间码（Longitudinal Time Code，LTC）的来源。

许多 DAW 都能和 Dolby Atmos Renderer 配合使用。在本课程中，将讨论 Dolby Atmos Renderer 和 Pro Tools Ultimate 两者的结合使用。Pro Tools Ultimate 是第一款支持和 Dolby Atmos Renderer 配套使用的 DAW，在音频后期制作和音乐制作环境中都占有重要的市场份额，其他 DAW 以及相应的配置会在第 13 课中讨论。

官方建议 DAW 最好能处理传递 128 轨的音频信息给 Dolby Atmos Renderer。尽管创作杜比全景声内容不是一定要用到 128 个输入，但支持这么多轨数确实能保证所需的灵活性，确保在不同的 DAW 和工作室之间得以移动项目。

另外，高速的存储也是需要考虑的因素。虽然大多数轨道量较大的工程都会要求高速存储，但是，如果 Renderer 是运行在 DAW 内部，并且重渲染音频是录到同一个 DAW 里的话，就需要更加注意存储速度和容量。

此外，专用的控制台也值得考虑。虽然有混音师可以用鼠标就出色地完成工作，但在处理沉浸式音频时，专用的控制台可以提供更高的效率。市面上有各种价格的控制台，专用于 Pro Tools 的也有很多。

系统时钟和同步

如果一个工作室是在专门的 Rendering & Mastering 工作站上运行 Dolby Atmos Renderer，那就要求有一个主系统时钟（Master Clock，MC）。这个主时钟需要是一个 Word Clock（WC） 发生器，能够分布式输出信号到 DAW 的音频接口和 Render 硬件。对于 Dante 系统来说，如果网络没有和同源的 PTP 时钟同步，那么就需要一个单独的设备来锁定 Word Clock。此外，设备间应避免使用菊花式拓扑连接（即用多个设备连接 Word Clock 而不是从信号源直接分发）。

系统时钟需要具备稳定性，避免由于时间码和采样点之间的差异而导致 Dolby Atmos Renderer 脱离录制状态。

如果工作室采用的是将 Dolby Atmos Renderer 运行在和 DAW 同一设备的方式，那么就不必需一个专门的主系统时钟，而是视接口和音频系统配置决定。

Dolby Atmos Renderer 跟随 DAW 发出的纵向时间码。纵向时间码可以来源于带有纵向时间码生成器的同步接口，比如 Avid Sync HD，也可以来源于 Dolby LTC Generator（纵向时间码生成器）插件。

音频信号和纵向时间码之间的延迟数值会因为 Renderer 和硬件配置的不同而有很大差别。对于每种配置，Renderer 中都有可用的预置，并且也可以在 Renderer 软件中调节延迟数值。

Dolby Atmos 的监听和监听控制器

混音面向家庭用户的 Dolby Atmos 内容时，推荐使用 7.1.4 的扬声器布局。四只天空通道的扬声器需要是真正的头顶扬声器，而不是放在地面的向上发声的扬声器（因为在做过声学处理的工作室中，向上发声的扬声器的信号会被天花板上的声学面板吸收一部分）。扩展的扬声器布局，比如 9.1.4，9.1.6 或者更高配置都是可以的，扬声器阵列也可以正常地适配到 Dolby Atmos 环境中。

注：扬声器摆位和测量会在单独的文档中详细解释。

Dolby Atmos Renderer中内置的 Binaural renderer 可以让我们通过耳机来监听，方便我们检查混音，快速地开始工作。当要交付成品时，应该首先使用扬声器来检查混音。

Dolby Atmos Renderer 中可以实现扬声器的衰减、mute 和 solo 功能，然而，专门的支持 7.1.4 配置的监听控制系统能够让我们更好地控制扬声器系统，并且还具有着完整的 B-Chain 功能以及信号源切换功能。

视频播放

在影视后期工作中，要求通过专门的接口来回放DAW 中的视频。许多 PCIe 或者雷电接口都是可用的。
