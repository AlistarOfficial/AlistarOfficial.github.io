Dolby Atmos Renderer 的核心功能

Dolby Atmos Renderer 有以下九个核心功能：

  1. 录制与纵向时间码同步的音床 / 声音对象及对象元数据信息，输出为 Master File Set 母带文件集。
  2. 在 Master Files 中包含了适用于 Dolby Digital Plus JOC 编码的 Downmix / Trim 元数据，以及适用于编码及监听双耳音频的元数据。
  3. 可以将 Master Files 导出为不同的夹层格式：ADM BWF（.wav）以及 IMF IAB（.mxf），可以全长导出或者编辑头尾后导出。
  4. 可以导出 Master Files 为 .mp4 格式，便于在消费级重放设备上检查成品质量。
  5. 可以基于 5.1 重渲染音频进行实时或离线的响度测量，并导出结果或者详细日志。
  6. 能够选择是否应用空间编码模拟来监听渲染到附加扬声器配置上的杜比全景声音乐或者 Master Files。
  7. 能够监听重渲染后的 Downmix 音频：7.1，5.1 以及立体声（或者自定义规格的音频，如 5.1.2）
  8. 能够用耳机监听立体声信号或者双耳重渲染音频。
  9. 能够实时监听或者离线导出重渲染内容。重渲染内容可以是完整的混音，如果使用了多音床输入，并且自定义了分组的话，重渲染内容也可以是单独的 Stem。
