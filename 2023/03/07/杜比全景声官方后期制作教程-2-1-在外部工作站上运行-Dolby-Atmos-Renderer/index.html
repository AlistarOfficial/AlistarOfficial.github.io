Dolby Atmos Renderer 是一款广泛应用于音乐、电影和游戏领域的音频处理软件，其能够在不同的硬件平台上实现音频的渲染和母带制作。本文将重点介绍该软件在 Rendering And Mastering Workstations 上的应用，包括基于 MADI 和 Dante 协议的不同配置，以及配套使用的数字音频工作站和相关外围设备等。对于音频制作工作者和相关领域的从业者而言，了解 Dolby Atmos Renderer 在不同平台和配置下的应用和优势，将有助于提高工作效率和音频制作质量。

翻译：Alistar

原文：Dolby Atmos Music 教程

一. 简介

在外部工作站运行的 Dolby Atmos Renderer 是 Dolby Atmos Renderer 的旗舰配置，需要使用 Dolby Atmos Mastering Suite 授权。这套配置推荐用于复杂的 Dolby Atmos 内容创作，因为它把渲染器和 DAW 需要的 CPU 资源分到两台电脑上负荷，并且还可以让多个数字工作站同时作为 Dolby Atmos Renderer 的信号源，以便在各个工作站中独立地进行预混音或者编辑工作。

外部运行的 Rendering and Mastering 工作站（以下简称 RMW）需要从 Dolby 经销商处购买，到手之前会预配置及测试所有需要的硬件和软件。

二. 使用 MADI 的 RMW 工作站

RMW 使用 MADI（多通道音频接口，Multi Channel Audio Interface）和 DAW 间往来数字音频信号，主要有三种配置：两种基于 Windows 的配置和一种基于 Mac 的配置。

MADI 配置 1：RME HDSPe 及 RME MADI FX

主要组件

  1. 在自定义的 Dell 7920 工作站上运行的Dolby Atmos RMW 包含以下组件：
  • RME HPSPe 配置
    • 预装两张 RME HDSPe MADI 卡，用于 128 个通道的 I / O
    • 预装一张 RME TCO 卡 用于 LTC 纵向时间码 及时钟输入
    • 安装 Dolby Atmos Renderer 软件，并用提供的 Dolby Atmos Mastering Suite License 进行授权
  • RME MADI FX 配置
    • 预装一张 MADI FX 卡，用于至少 128 个通道的 I / O
    • 安装 Dolby Atmos Renderer 软件，并用提供的 Dolby Atmos Mastering Suite License 进行授权
  1. 运行支持软件的 DAW 系统包含以下组件：
  • 原生支持 Atmos Panner 并且能够直连 Renderer 的 DAW，或者能够使用 Dolby Atmos Music Panner 插件的 DAW
  • 一台连接 128 个通道 MADI I / O 的硬件
  • Dolby Atmos Renderer 的远程控制软件（包含在 Renderer 的安装器里）
    • 远程控制软件可以选择安装在单独的电脑上（可以基于 Windows 或 Mac），来节省运行 DAW 这台电脑的屏幕空间
  1. 一台同步外围设备，用于为 RMW 生成纵向时间码，并将 DAW 的时钟与 RMW 的 Word Clock 同步

MADI 配置 2：RME MADIface XT

主要组件

  1. 一台 Dolby Atmos RMW，可以是专门的自定义的 Windows Dell 7920 或者经过认证的 Mac OS 配置。该工作站包含以下组件：
  • RME MADIface XT USB 3.0 接口
  • 安装 Dolby Atmos Renderer 软件，并用提供的 Dolby Atmos Mastering Suite License 进行授权
  1. 运行支持软件的 DAW 系统包含以下组件：
  • 原生支持 Atmos Panner 并且能够直连 Renderer 的 DAW，或者能够使用 Dolby Atmos Music Panner 插件的 DAW
  • 一台连接 128 个通道 MADI I / O 的硬件
  • Dolby Atmos Renderer 的远程控制软件（包含在 Renderer 的安装器里）
    • 远程控制软件可以选择安装在单独的电脑上（可以基于 Windows 或 Mac），来节省运行 DAW 这台电脑的屏幕空间
  1. 一台同步外围设备，用于为 RMW 生成纵向时间码，并将 DAW 的时钟与 RMW 的 Word Clock 同步。或者也可以在 DAW 中运行 Dolby LTC Generator 插件来生成纵向时间码

三. 使用 Dante I / O 的 RMW 工作站

Dolby Atmos Mastering Suite 3.0 版本支持 RMW 使用 Dante 协议进行音频信号的输入输出，RMW 可以基于 Windows 或 Mac 系统。基于 Dante 协议的工作站配置在两个平台上都差不多。

比起 MADI，基于 Dante 的 RMW 需要的连接更少，但路由和配置会更复杂。如果是首次使用 Dante 系统，推荐观看 Audinate 提供的“Getting Started with Dante”和“Dante Controller 101”系列视频：

https://www.audinate.com/learning/training-certification/video-tutorials

主要组件

  1. 一台 Dolby Atmos RMW，可以是专门的自定义的 Windows Dell 7920 或者经过认证的 Mac OS 配置。该工作站包含以下组件：
  • 预装的Focusrite RedNet PCIeR Dante 卡（如果工作站电脑是 Mac Mini 或者 Mac Pro 6，就需要用认证的 PCIe 雷电扩展机架，根据 Mac 的不同型号，可以是雷电 2 或者雷电 3。）
  • 安装 Dolby Atmos Renderer 软件，并用提供的 Dolby Atmos Mastering Suite License 进行授权
  • 一个 Audinate 的模拟转 Dante 转换器，用于纵向时间码。
  1. 一个 Mac 系统下的运行支持软件的 DAW。包含以下组件：
  • 原生支持 Atmos Panner 并且能够直连 Renderer 的 DAW，或者能够使用 Dolby Atmos Music Panner 插件的 DAW
  • 一台连接 Dante I / O 的硬件
  • Dolby Atmos Renderer 的远程控制软件（包含在 Renderer 的安装器里）
    • 远程控制软件可以选择安装在单独的电脑上（可以基于 Windows 或 Mac），来节省运行 DAW 这台电脑的屏幕空间
  1. 一台兼容 Dante 协议的网络交换机，交换机上的设备需要以星形拓扑结构连接（即每个设备都必须直连到交换机上）。
  2. 一台同步外围设备，用于为 RMW 生成纵向时间码，并将 DAW 的时钟与 RMW 的 Word Clock 同步。或者也可以在 DAW 中运行 Dolby LTC Generator 插件来生成纵向时间码
